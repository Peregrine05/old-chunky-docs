Minecraft Compatibility

Bedrock Edition worlds are not supported.

Most entities are currently not rendered by Chunky, and some special blocks can not be rendered either. Here is a list of the supported Minecraft versions in addtion to all the things that Chunky cannot render.

  Chunky Version Issue Tracker
  2.4.1  
Minecraft Java versions 1.2.1 - 1.17.1  
Mod blocks No #88, #426, #266
Custom block models No #88, #426, #266
Item frames No #790, #789
Held Item rendering No #669, #595
Players Partial #365
Mobs (animals and monsters) No #41
Book (enchanting table) Yes  
Beacon Beam Yes  
Ender Crystals No #41
Boats No  
Mine carts No  
Falling Sand Entities No #454
Particles No #41
